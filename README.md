# 🎵 Music Downloader

Инструмент для скачивания музыки через **Hitmo** по списку треков.

Проект позволяет обходить ограничения и цензуру музыкальных сервисов, скачивая треки по:

* ссылке на **плейлист или альбом из Яндекс Музыки**
* **текстовому файлу** со списком треков

Сервис автоматически находит треки на Hitmo и скачивает их.

---

## ✨ Возможности

* 📥 Скачивание треков по **ссылке на Яндекс Музыку**
* 📄 Поддержка **txt файлов** со списком треков
* 🔎 Автоматический поиск треков через **Hitmo**
* 📦 Архивация скачанной музыки
* ⚡ Быстрая и простая загрузка
* 🌍 Возможность обхода региональных ограничений

## 📂 Структура проекта

```
music-downloader
│
├── cmd/server            # точка входа
├── config                # конфигурация
│
├── internal
│   ├── api               # HTTP API
│   ├── domain            # бизнес логика

│   └── zip               # архивация треков
│
└── pkg/logger            # логирование
```

## 🚀 Установка

Требования:

* **Go 1.25**

### 1. Клонировать репозиторий

```bash
git clone https://github.com/yourname/music-downloader.git
cd music-downloader
```

### 2. Установить зависимости

```bash
go mod download
```

### 3. Запустить сервер

```bash
go run cmd/server/main.go
```

## 📥 Использование

### Скачивание по ссылке на Яндекс Музыку

Вставьте ссылку на:

* плейлист
* альбом

Пример:

```
https://music.yandex.ru/users/username/playlists/123
```

Сервис:

1. Получит список треков
2. Найдёт их на Hitmo
3. Скачает
4. Соберёт архив

---

### Скачивание по текстовому файлу

Создайте файл `tracks.txt`

```
Linkin Park - Numb
Rammstein - Sonne
Daft Punk - Harder Better Faster Stronger
```

Загрузите файл в сервис — треки будут автоматически найдены и скачаны.

## 💡 Идея проекта

Многие музыкальные сервисы ограничивают доступ к музыке из-за:

* региональных ограничений
* цензуры
* удаления контента

Этот проект создан как **технический инструмент для получения доступа к уже существующей музыке**, используя альтернативные источники.
